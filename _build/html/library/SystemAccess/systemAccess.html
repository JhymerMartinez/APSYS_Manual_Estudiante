2. Acceso al sistema¶

2.1. Acceso Inicial¶

El ingreso al sitio de administración se lo realiza mediante la URL: http://. El estudiante se debe logear ingresando su cédula y su contraseña en cada casilla correspondiente:

Ingreso al Sistema

Note

La contraseña es la misma utilizada para el ingreso al Sistema de Gestión Académica (SGA)

En caso de proporcionar datos incorrectos se presentará un mensaje correspondiente

Error de logueo

Caso contrario el se presenta un mensaje de logueo exitoso y automáticamente sera redirigido a la opción Seleccionar módulo

Ingreso correcto
