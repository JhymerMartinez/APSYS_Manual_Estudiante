1. Introducción¶

El presente Manual del Estudiante tiene como principal objetivo servir de guía para los estudiantes universitarios que se beneficiarán del uso del sistema para la predicción del desempeño académico, el mismo que busca facilitar el proceso de seguimiento y progreso académico

El manual está dividido en secciones claramente definidas con imágenes y explicaciones que permitan al usuario solventar cualquier duda o inconveniente que se pueda presentar al momento del uso del sistema.
